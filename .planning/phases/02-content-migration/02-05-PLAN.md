**Step 1 — Fix alias collision in mdsvex.config.js:**

In the `aliases` object (around line 22-27), change ONLY the `caution` value:

```js
aliases: {
  danger: "deter",
  info: "note",
  tip: "commend",
  caution: "warn",     // WAS "deter" — now maps to "warn" (amber), distinct from danger's "deter" (red)
  warning: "warn",
},
```

This makes:
- `:::danger` -> `callout-deter` (red) — unique
- `:::caution` -> `callout-warn` (amber) — now distinct from danger
- `:::warning` -> `callout-warn` (amber) — same as caution, which is correct
- `:::info` -> `callout-note` (blue) — unique
- `:::tip` -> `callout-commend` (green) — unique

**Step 2 — Add callout CSS rules to article.css:**

Append the following block at the end of article.css (after the `@media (max-width: 359px)` block, before the final closing). These rules target the raw HTML output of remark-callout-directives:

```css
/* ============================================================
   CALLOUT / ADMONITION BLOCKS
   Output of remark-callout-directives: <aside class="callout callout-{type}">
   Types: deter (danger), note (info), commend (tip), warn (caution/warning)
   ============================================================ */

.vsa-article .callout {
  border-left: 3px solid var(--vsa-border);
  border-radius: var(--vsa-radius-sm);
  padding: var(--vsa-space-md);
  margin: var(--vsa-space-md) 0;
}

/* Indicator row: icon + title */
.vsa-article .callout-indicator {
  display: flex;
  align-items: center;
  gap: var(--vsa-space-xs);
  margin-bottom: var(--vsa-space-sm);
  font-weight: 600;
}

/* SVG icon sizing */
.vsa-article .callout-hint svg {
  width: 18px;
  height: 18px;
  flex-shrink: 0;
}

/* Content body */
.vsa-article .callout-content {
  font-size: var(--vsa-font-size-base);
}

/* Remove default paragraph margin inside callout content */
.vsa-article .callout-content p:last-child {
  margin-bottom: 0;
}

/* --- Type: deter (:::danger) — Red --- */
.vsa-article .callout-deter {
  border-left-color: var(--vsa-destructive);
  background: var(--vsa-danger-bg);
}

.vsa-article .callout-deter .callout-indicator {
  color: var(--vsa-destructive);
}

/* --- Type: warn (:::caution, :::warning) — Amber --- */
.vsa-article .callout-warn {
  border-left-color: var(--vsa-warning);
  background: var(--vsa-warning-bg);
}

.vsa-article .callout-warn .callout-indicator {
  color: var(--vsa-warning);
}

/* --- Type: note (:::info) — Blue --- */
.vsa-article .callout-note {
  border-left-color: var(--vsa-info);
  background: var(--vsa-info-bg);
}

.vsa-article .callout-note .callout-indicator {
  color: var(--vsa-info);
}

/* --- Type: commend (:::tip) — Green --- */
.vsa-article .callout-commend {
  border-left-color: var(--vsa-accent);
  background: var(--vsa-accent-subtle);
}

.vsa-article .callout-commend .callout-indicator {
  color: var(--vsa-accent);
}
```

Do NOT modify or remove Admonition.svelte — it remains dead code for now (a future refactor could wire it up, but that is out of scope for this gap closure).
