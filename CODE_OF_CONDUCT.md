# Código de Conducta / Code of Conduct

## Nuestro compromiso / Our Pledge

Este proyecto existe para construir un futuro mejor para Venezuela basado en datos, transparencia y participación. Nos comprometemos a crear un espacio libre de acoso, discriminación y toxicidad.

This project exists to build a better future for Venezuela based on data, transparency and participation. We pledge to make it a harassment-free, discrimination-free, toxicity-free space.

## Estándares / Standards

**Comportamiento esperado:**
- Usar lenguaje respetuoso e inclusivo
- Aceptar críticas constructivas con madurez
- Priorizar datos sobre opiniones
- Respetar perspectivas diferentes sobre la situación venezolana

**Comportamiento inaceptable:**
- Propaganda política de cualquier bando
- Ataques personales, insultos o amenazas
- Desinformación deliberada
- Discriminación por nacionalidad, estatus migratorio, género, raza, orientación sexual o cualquier otra razón
- Doxxing o publicación de información privada

## Aplicación / Enforcement

Los maintainers del proyecto tienen el derecho y la responsabilidad de remover, editar o rechazar comentarios, commits, código, issues y otras contribuciones que violen este código.

Violaciones pueden reportarse a: conduct@venezuela-sa.org

## Atribución

Basado en el [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
