# Inversión Ciudadana Universal: Todos Somos Socios

> Artículo 1, ICCPR (ONU, 1966): “Todos los pueblos pueden disponer libremente de sus riquezas y recursos naturales.” La palabra clave es **pueblos**, no Estados.

## 4 Capas de Participación

1. **Dividendo Automático** — Para TODOS. Sin inversión requerida. Modelo: [Alaska PFD USD 1.000/persona (2025)](https://pfd.alaska.gov/)
2. **Bonos Ciudadanos** — Desde USD 10. Interés fijo + participación
3. **Fondo Tech Venezuela** — 20–30% del fondo invierte en tech interno (startups, data centers, IA). Modelo VC nacional
4. **Participaciones VIN** — Desde USD 500. Visión IPO años 8–12
